2. App Description

Arcmark is a macOS application that helps you manage, search, and organize bookmarks and browsing history from Safari and Chrome. The App reads browser data locally, allows you to tag and annotate items, and exports your data in various formats.

The App is developed and operated by Stefan Godoroja.

3. Free and Pro Features

3.1 Free Features

The free version of Arcmark includes:

  • Import bookmarks and history from Safari and Chrome
  • Search across all your browser data
  • Up to 10 unique tags
  • Up to 3 collections
  • Basic filtering (by browser, by type)
  • Export to HTML format

3.2 Arcmark Pro

Arcmark Pro unlocks additional features:

  • Unlimited tags - Organize without limits
  • Unlimited collections - Create as many as you need
  • Multi-profile Chrome import - Access all Chrome profiles
  • Advanced filters - Duplicates, Content Type, Top Visits, Smart Filter
  • Export formats - CSV, JSON, Markdown, OPML, Plain Text
